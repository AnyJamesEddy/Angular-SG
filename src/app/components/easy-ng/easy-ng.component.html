What you see is called string interpolation and it is one of the three main ways to make your Typescript code work with your HTML template!

These communication methods are:
  • String Interpolation
  • Property Binding
  • Event Binding

If you click on Button 1, it will become green, while Button 2 will become red and viceversa. This has been made via event binding.

Here's the snippet: 

  <p  style="font-style: italic">
    Consider now these buttons:
  <button
        [style.background-color]="b1Pressed ? 'green' : (b2Pressed ? 'red' : null)" class="myButton1"
        (click)="onClick()">BUTTON 1
  </button>
  -
  <button 
        [style.background-color]="b2Pressed ? 'green' : (b1Pressed ? 'red' : null)"
        (click)="onClick()">BUTTON 2
  </button>
  </p>
